# K-Study Bridge

K-Study Bridge нь Солонгост суралцах гадаад оюутнуудад зориулсан мэдээллийн платформ юм. Энэ вэб сайт нь их сургууль, виз, орон сууц, амьдралын зардал болон зөвлөгөөг нэг дороос хүргэдэг.

## 📁 Файлын бүтэц

- `index.html` – Нүүр хуудас
- `service-university.html` – Их сургуулийн мэдээлэл
- `service-visa.html` – Визний зөвлөгөө
- `service-housing.html` – Байр болон орон сууцны зөвлөгөө
- `service-living.html` – Солонгост амьдрах, бүртгүүлэх, утас, банк зэрэг мэдээлэл
- `css/style.css` – Загварын файл
- `js/app.js` – Гол контроллер, хэл, сэдвийн горим, интерактив табууд, FAQ, витбот
- `js/universities.js` – Их сургуулийн өгөгдөл ба хайлтын логик
- `js/calculator.js` – Амьжиргааны зардлын тооцоолол
- `js/booking.js` – Зөвлөгөө захиалах формын алхаа алхмаар боловсруулалт, draft хадгалах болон илгээх статустай
- `js/chatbot.js` – Ji-Woo чатботын логик
- `images/` – Веб сайтын зургууд

## 🚀 Гол онцлогууд

- Монгол, Англи, Солонгос хэлний локализаци дэмжлэгтэй
- Сургуулийн хайлт, байршил, төлбөр, TOPIK шаардлагыг харуулах интерактив панель
- Амьжиргааны зардал тооцоолох интерактив слайдер
- Визний шаардлагын чеклист болон явцын прогресс мөр
- Нэг алхамтай зөвлөгөө захиалах Wizard форм
- Чатбот жинхэнэ AI түлхүүртэй ажиллах боломж болон офлайн fallback логик
- Хүчтэй хялбархан зассан хүртээмжтэй байдал (tab-focus, ARIA сайжруулалт)

## 🧪 Локал дээр ажиллуулах

1. Файлыг шууд нээх:
   - `index.html` файлыг хялбархан браузерээр нээж үзнэ.

2. Сервэр ашиглах (зөвлөмж):
   - `Live Server` өргөтгөл ашиглан `index.html` файлыг нээх
   - Эсвэл терминал дээр `python -m http.server 5500` гэж ажиллуулаад `http://localhost:5500` руу нэвтрэнэ.

## ♿ Хандалт ба оновчлол

- `service-card` элементүүдийг клавиатураар `Enter`/`Space`-аар идэвхжүүлэх боломжтой болгов.
- FAQ accordion-ийг `role="button"` болон `tabindex="0"`-той болгож, Enter/Space-ээр хааж/нээх боломжтой болголоо.
- Навигацийн `dropdown` товчоор нээх/хаахыг `aria-expanded`-тэй болгож, гадагш дарахад болон Esc товч дарахад хаагдах support нэмэв.
- Навигацийн холбоосуудыг одоогийн хуудас эсвэл хэсэгт тулгуурлан тодруулж, хэрэглэгчдэд хэд дэх хэсэг дээр байгааг илүү тод харуулжээ.
- Хавтасны навигацид `role="navigation"`, `aria-controls`, `aria-label` зэрэг semantics нэмж, нэмэлт хүртээмжийг сайжруулсан.
- Хэрвээ хэрэглэгч хуудсыг доош гүйлгэвэл header илүү тодорхой харагдаж, навигацийн толгой илүү тодорно.
- `button`, `input`, `select`, `textarea` элементүүдэд `:focus-visible` хүрээ нэмэгдүүлж, анхаарал татсан хэлийг илүү тодруулсан.
- Чатботын нээх товч дээр `aria-expanded` нэмэн, чат мессеж бие хариу өгөх хэсгийг `aria-live="polite"` болгосон.

## ☁️ Deployment

Энэ нь статик вебсайт тул Vercel болон бусад статик хост платформ дээр шууд байршуулж болно.

- Vercel дээр байрлуулах:
  1. `Kstudy-master/Kstudy-master` хавтасийг репозиторийн үндсэн фолдер болго.
  2. Vercel-д шинэ проект үүсгэн, GitHub холбож, платформыг статик сайт болгон тохируул.

## 💡 Тэмдэглэл

- Хэрэглэгчийн хэлний тохиргоог `localStorage` ашиглан хадгалдаг.
- Чатботыг Google Gemini API түлхүүрээр идэвхжүүлж, түлхүүр оруулалгүй бол офлайн fallback горим ажиллана.
