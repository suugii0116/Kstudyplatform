💼 Үнийн багц

Таны хэрэгцээнд нийцсэн үйлчилгээний үнэ

Сургалтын, визний, байрны болон амьдралын дэмжлэгийн багцуудын онцлох давуу тал, төлбөрийн нөхцөл, үед тохирсон шийдлүүд.

Үйлчилгээний багцууд

Та өөрийн хэрэгцээ, төсөвт хамгийн сайн нийцэх багцыг сонгоно уу. Бид бүх үйл явцад тантай хамт байх болно.

Basic

Бүх зүйлээ өөрөө бэлдэж буй ч мэргэжлийн хяналт шаардлагатай оюутнуудад зориулав.

₮250,000 (Нэг удаагийн төлбөр)
  • ✓ Сургуулийн зөвлөгөө (1 цаг)
  • ✓ Эсээ болон CV хяналт
  • ✓ Визний бичиг баримтын чеклист
  • ✓ 1 сарын турш и-мэйлээр зөвлөгөө өгөх
  • ✗ Визний бичиг баримт шалгах
  • ✗ Орон байр хайх үйлчилгээ
ХАМГИЙН ЭРЭЛТТЭЙ

Professional

Сургууль болон визний материалыг бүрэн хариуцаж гүйцэтгэх цогц үйлчилгээ.

₮750,000 (Нэг удаагийн төлбөр)
  • ✓ Сургууль сонголт, элсэлтийн баталгаа
  • ✓ Эсээ, материалыг бүрэн бэлтгэх
  • ✓ ЭСЯ-ны визний ярилцлагын бэлтгэл
  • ✓ Орон сууц хайх үйлчилгээ
  • ✓ ЭСЯ-ны ярилцлагад бэлтгэх (Zoom)
  • ✓ Орон байр хайх үндсэн туслалцаа
  • ✓ 3 сарын турш 24/7 зөвлөгөө, дэмжлэг

Premium

Эхнээс нь дуустал бүх зүйлийг бид хариуцаж, Солонгост очих хүртэлх VIP дэмжлэг.

₮1,500,000 (Нэг удаагийн төлбөр)
  • ✓ Professional багц бүрэн багтсан
  • ✓ Нисэх онгоцны буудлаас тосох
  • ✓ Банкны данс, утас, ARC нээхэд туслах
  • ✓ Эхний 1 сарын хувийн менторшип
  • ✓ Орон байрны дээд зэрэглэлийн сонголтууд
  • ✓ Гадаад иргэний үнэмлэх (ARC) захиалах
  • ✓ Банкны данс нээх, утас авах үйлчилгээ
  • ✓ 6 сарын турш бүх төрлийн зөвлөгөө, дэмжлэг
  • ✓ Сөүлд амьдрах дасан зохицох тусгай сургалт

❓ Түгээмэл асуултууд

Төлбөрийг хэд хувааж төлж болох вэ?

Premium багцын төлбөрийг 2 хувааж төлөх боломжтой. Эхний тал нь гэрээ байгуулах үед, үлдсэн тал нь сургуульд тэнцсэний дараа төлөгдөнө.

Багцын үнэд сургуулийн бүртгэлийн хураамж багтсан уу?

Үгүй. Сургуулийн бүртгэлийн хураамж (Application fee) болон ЭСЯ-ны хураамжууд үйлчилгээний төлбөрт багтаагүй болно.

Аль багцыг сонгох нь зүйтэй вэ?

Ихэнх оюутнууд "Professional" багцыг сонгодог. Энэ нь үнэ болон үйлчилгээний хамгийн зөв, тэнцвэртэй сонголт юм.

Үйлчилгээний хугацааг сунгах боломжтой юу?

Тийм ээ, хэрэв нэмэлт зөвлөгөө шаардлагатай бол үйлчилгээг сунгах багц (₩200,000 / 3 сар) авч болно.
